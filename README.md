# Vue-node-超市系统
用户端点击商品可进入商品的评论页面，用户个人信息不完整可能生成订单失败，购物车页面修改数量时没写库存验证，如果折扣价格没显示刷新即可，点击订单页面的用户名可进入订单详情。
